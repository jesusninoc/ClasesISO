# Monitorización

- OxiPulse is an ultralight, open source server monitoring agent written in Rust. It collects your server's vital signs — CPU, RAM, disk, and network — and ships them to any OpenTelemetry-compatible backend using the OTLP protocol https://github.com/securyblack/oxi-pulse 
- Monitorización en PowerShell

| Comando | Qué monitoriza | Uso básico | Qué muestra |
|---|---|---|---|
| `Get-Counter '\PhysicalDisk(_Total)\% Disk Time'` | Disco | `Get-Counter '\PhysicalDisk(_Total)\% Disk Time'` | Actividad de disco |
| `Get-Counter '\Network Interface(*)\Bytes Total/sec'` | Red | `Get-Counter '\Network Interface(*)\Bytes Total/sec'` | Tráfico de red |
| `Get-WinEvent -LogName System -MaxEvents 20` | Logs | `Get-WinEvent -LogName System -MaxEvents 20` | Eventos recientes |
| `Get-NetAdapter` | Red | `Get-NetAdapter` | Estado de interfaces |
| `Get-NetTCPConnection` | Conexiones | `Get-NetTCPConnection` | Conexiones TCP activas |
| `resmon` | Monitor gráfico | `resmon` | Abre el monitor de recursos |
| `perfmon` | Monitor gráfico | `perfmon` | Abre el monitor de rendimiento |
