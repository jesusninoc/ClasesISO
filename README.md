> Repositorio educativo con **clases, ejercicios y proyectos sobre implantación de sistemas operativos**, administración de sistemas, redes, automatización y scripting.

Este repositorio recopila **material docente real utilizado en clases**, organizado cronológicamente y con ejemplos prácticos sobre administración de sistemas.

---

# Vista general

**ClasesISO** es una colección extensa de contenidos utilizados para enseñar:

- administración de sistemas
- redes
- scripting
- seguridad básica
- automatización de tareas
- análisis de sistemas

Los contenidos están estructurados en **archivos Markdown por fecha**, cada uno correspondiente a una sesión de clase o conjunto de ejercicios.

Esto permite seguir fácilmente la evolución del curso.
