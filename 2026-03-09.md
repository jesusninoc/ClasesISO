- Comandos y cmdlets
  - Monitorización en Linux

| Comando | Qué monitoriza | Uso básico | Qué muestra |
|---|---|---|---|
 `iostat` | Disco y CPU | `iostat -xz 1` | Uso de disco, lecturas, escrituras, espera |
| `iotop` | I/O por proceso | `sudo iotop` | Qué procesos están usando más disco |
| `df` | Espacio en disco | `df -h` | Espacio usado y libre por sistema de archivos |
| `du` | Uso de disco por carpeta | `du -sh /ruta` | Tamaño de un directorio |
| `uptime` | Carga del sistema | `uptime` | Tiempo encendido y carga media |

  - Monitorización en PowerShell

| Comando | Qué monitoriza | Uso básico | Qué muestra |
|---|---|---|---|
| `Get-Process` | Procesos | `Get-Process` | Lista de procesos activos |
| `Get-Process \| Sort-Object CPU -Descending \| Select-Object -First 10` | Procesos por CPU | `Get-Process \| Sort-Object CPU -Descending \| Select-Object -First 10` | Procesos que más CPU consumen |
| `Get-Service` | Servicios | `Get-Service` | Estado de los servicios |
| `Get-Counter '\Processor(_Total)\% Processor Time'` | CPU | `Get-Counter '\Processor(_Total)\% Processor Time'` | Uso de CPU |
| `Get-Counter '\Memory\Available MBytes'` | Memoria | `Get-Counter '\Memory\Available MBytes'` | RAM disponible |

- Ejercicios
```Bash
#!/bin/bash

echo "=============================="
echo " MONITORIZACIÓN BÁSICA LINUX "
echo "=============================="
echo

# Fecha y hora
echo "Fecha: $(date)"
echo

# Uptime
echo "Tiempo encendido:"
cat /proc/uptime | cut -d' ' -f1 | awk '{print $1 " segundos"}'
echo

# Carga media
echo "Carga media (1, 5 y 15 min):"
cat /proc/loadavg | cut -d' ' -f1-3
echo
```
