# Monitorización

| Comando | Qué monitoriza | Uso básico | Qué muestra |
|---|---|---|---|
| `Get-WinEvent -LogName System -MaxEvents 20` | Logs | `Get-WinEvent -LogName System -MaxEvents 20` | Eventos recientes |
| `Get-NetAdapter` | Red | `Get-NetAdapter` | Estado de interfaces |
| `Get-NetTCPConnection` | Conexiones | `Get-NetTCPConnection` | Conexiones TCP activas |
| `Test-Connection google.es -Count 4` | Conectividad | `Test-Connection google.es -Count 4` | Prueba de red |
| `Get-Volume` | Discos | `Get-Volume` | Estado y espacio de unidades |
| `Get-PSDrive` | Unidades | `Get-PSDrive` | Espacio en discos |
| `resmon` | Monitor gráfico | `resmon` | Abre el monitor de recursos |
| `perfmon` | Monitor gráfico | `perfmon` | Abre el monitor de rendimiento |

- Ejercicios
```Bash
#!/bin/bash

# Tráfico de red (bytes recibidos y enviados)
echo "Tráfico de red:"
cat /proc/net/dev | tail -n +3 | while IFS= read -r linea; do
    interfaz=$(echo "$linea" | cut -d':' -f1 | tr -d ' ')
    datos=$(echo "$linea" | cut -d':' -f2)
    rx=$(echo "$datos" | awk '{print $1}')
    tx=$(echo "$datos" | awk '{print $9}')
    echo " - $interfaz | RX: $rx bytes | TX: $tx bytes"
done
echo

# Usuarios conectados
echo "Usuarios conectados:"
who
echo
```

# Resolución incidencias
- Repaso de AD (GPO).
- En un Windows Server, instala el rol de WSUS. Configúralo para que sincronice las actualizaciones de Windows 10. Luego, crea una GPO y configúrala para que los clientes de tu dominio apunten a este servidor WSUS para buscar actualizaciones. Aplica la GPO a una OU con equipos cliente y fuerza la detección de actualizaciones en uno de ellos.
- En una máquina virtual con Windows, habilita el Escritorio Remoto. Desde tu PC anfitrión, abre el cliente de Conexión a Escritorio Remoto (mstsc.exe) e introduce la dirección IP de la máquina virtual para conectarte a ella. Experimenta la sensación de controlar la VM como si estuvieras sentado delante de ella.
