# Resolución incidencias
- Añadir un equipo al Windows Server.
- Repaso de AD (GPO).
- En un Windows Server, instala el rol de WSUS. Configúralo para que sincronice las actualizaciones de Windows 10. Luego, crea una GPO y configúrala para que los clientes de tu dominio apunten a este servidor WSUS para buscar actualizaciones. Aplica la GPO a una OU con equipos cliente y fuerza la detección de actualizaciones en uno de ellos.
  - Configuración del equipo → Plantillas administrativas → Componentes de Windows → Windows Update (http://TU_SERVIDOR_WSUS:8530)

# Volumen vs partición

## Windows

- Partición (Windows): División del disco definida en la tabla de particiones (MBR o GPT). Es una estructura a bajo nivel que organiza el almacenamiento físico.
- Volumen (Windows): Unidad lógica que usa el sistema operativo (C:, D:, etc.), con sistema de archivos y accesible para el usuario. Puede coincidir con una partición o abarcar varias. https://learn.microsoft.com/es-es/windows-server/storage/disk-management/manage-basic-volumes
- Diferencia clave: La partición pertenece al nivel físico/estructural del disco, mientras que el volumen es la representación lógica y utilizable por Windows.

## Linux

- Partición (Linux): División del disco físico definida en la tabla de particiones (MBR/GPT), representada como dispositivos tipo /dev/sda1, /dev/sda2, etc. https://help.ubuntu.com/stable/ubuntu-help/disk-partitions.html.es
- Volumen (Linux – LVM): Unidad lógica creada sobre una o varias particiones o discos dentro de Logical Volume Manager (LVM). Es flexible, redimensionable y no depende directamente de una sola partición.
- Diferencia clave: La partición es estructura física del disco, mientras que el volumen (LVM) es una abstracción lógica que permite agrupar, ampliar o modificar el almacenamiento sin depender de la estructura física inicial.
